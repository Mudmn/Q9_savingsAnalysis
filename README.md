# Q9_savingsAnalysis
Assignment question 2026.

 A financial consultant wants to study the savings habits of Malaysians citizens. The program should first ask the user how many participants data will be entered. For each participant, the user
 must input gender, age and savings. The program should store and display all the data, calculate the total and average savings, and then identify and display all participants whose age are older
 than minimum age entered by user, showing their gender, age and savings. If no participants are older than the minimum age, the program should display an appropriate messagesuch as "No participant
 is older than .... years old".
